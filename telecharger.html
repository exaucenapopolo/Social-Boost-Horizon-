Comment installer l'application ?
En 3 étapes simples sur votre téléphone Android
1
Téléchargez le fichier APK Cliquez sur le bouton "Télécharger l'application" ci-dessus. Le fichier SocialBoostHorizon.apk sera téléchargé sur votre téléphone.
2
Autorisez l'installation Allez dans Paramètres → Sécurité → Sources inconnues (ou "Installer des applications inconnues") et activez l'option pour votre navigateur ou gestionnaire de fichiers.
3
Ouvrez le fichier et installez Trouvez le fichier APK dans vos téléchargements, touchez-le et suivez les instructions d'installation. L'application apparaît sur votre écran d'accueil.
L'application est disponible uniquement sur Android pour l'instant. La version iOS (iPhone) est en cours de développement.
